📖 项目简介
本项目是一款基于 STC15系列51单片机 和 SG90舵机 开发的红外遥控智能开关。它主要用于解决非智能灯具的远程控制需求，通过机械臂模拟人工拨动开关。
核心特色：
全场景适配：配合不同的 3D 打印机械臂附件，可安装在拨杆开关、按键开关或触摸开关上。
极致省电：通过单片机深度睡眠模式与舵机电源物理切断技术，实测 3.7V 锂电池待机时长高达 37 天。
精准控制：采用外部中断+定时器捕获红外信号，解码稳定，响应迅速。
🛠️ 硬件架构
1. 核心控制单元
MCU: STC15W408AS (高速度、低功耗、无需外部晶振)。
执行机构: SG90 微型舵机 (0-180° 精准控制)。
输入模块: 红外接收头 (1838B)，支持通用 NEC 协议红外遥控器。
2. 电源与功耗管理 (本项目亮点)
舵机断路逻辑：使用 P-MOS 管 (AO3401) 作为舵机电源开关。舵机仅在动作瞬间通电，平时处于完全断电状态，彻底杜绝舵机静态抖动和空载功耗。
充电方案：集成 TP4056 锂电池充电管理电路，支持 Type-C 接口便捷充电。
MCU 节能：软件引入 Power-down 模式，由红外接收端的外部中断唤醒。
💻 软件设计
1. 红外解码算法
项目摒弃了传统的查询等待法，采用外部中断 (INT) 捕捉红外波形的下降沿，并结合定时器 (Timer) 记录脉冲宽度：
判断引导码 (9ms Low + 4.5ms High)。
区分数据位 "0" (560us Low + 560us High) 与 "1" (560us Low + 1.6ms High)。
通过校验指令集确保操作的准确性，防止误触发。
2. 状态机控制逻辑
code
C
// 伪代码逻辑预览
if (IR_Data_Ready) {
    Match_Command();       // 比对指令集
    Power_On_Servo();      // 开启舵机电源 (MOS管导通)
    Execute_Movement();    // 执行旋转动作
    Power_Off_Servo();     // 关闭舵机电源 (MOS管截止)
    Enter_Sleep_Mode();    // MCU 进入掉电模式
}

🚀 如何使用
硬件连接：根据 Hardware 目录下的原理图焊接 PCB。
固件烧录：使用 STC-ISP 工具将十六进制文件下载至 STC15W408AS。
对码：按下遥控器对应按键，记录其十六进制代码并更新至代码中的 Command_List。
安装：根据目标开关类型，选择并安装合适的 3D 打印支架，使用 3M 胶固定在开关旁。
